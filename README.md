# CadaTokenV2

## Installatoion

```bash
 git clone https://github.com/CanadaCoinProject/CadaTokenV2
 
 cd CadaTokenV2 
 
 npm i --force 
```

## Config

Rename secrets.example.js to secrets.js and replace the parameters 

## Deployment 

```bash 
  # Testnet ddeployment 
  
  #Kovan
  yarn deploy_kovan
  
  #Ropsten 
  yarn deploy_ropsten
  
  #Rekinby
  yarn deploy_rinkeby
  
  # Bsc testnet
  yarn deploy_bsc_testnet
  
  # Mainnet Deployment
  
  # Ethereum Mainnet
   yarn deploy_eth_mainnet
   
   # BSC Mainnet
   yarn deploy_bsc_mainnet
  
```
